## Technology

- Framework
  - [Next.js](https://github.com/vercel/next.js)
- Monorepo tool
  - [Turborepo](https://github.com/vercel/turbo)
- JavaScript Runtime
  - [Bun](https://github.com/oven-sh/bun)

## Quick Start

### Install dependencies

```sh
bun install
```

### Run locally

```sh
bunx turbo dev
```

- Web
  - https://localhost:3001/
- Auth
  - https://localhost:3002/
