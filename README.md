## Technology

- Framework
  - [Next.js](https://github.com/vercel/next.js)
- Monorepo tool
  - [Turborepo](https://github.com/vercel/turbo)
- Styling
  - [TailwindCSS](https://tailwindcss.com/)
- JavaScript Runtime
  - [Bun](https://github.com/oven-sh/bun)
- Linter, Formatter
  - [Biome](https://biomejs.dev/)

## Quick Start

### Install dependencies

```sh
bun install
```

### Run locally

```sh
bunx turbo dev
```

- Web
  - http://localhost:3001/
- Auth
  - http://localhost:3002/
